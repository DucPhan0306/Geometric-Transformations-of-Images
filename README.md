# Geometric-Transformations-of-Images
Library: OpenCV and Numpy


Transformations:
 - Scale
 - Rotate
 - Shear
 - Flip
 - Translate
 - Euclidean = Rotate + Translate
 - Similarity = Rotate + Translate + Uniform scale
 - Affine = Rotate + Translate + Uniform scale + Shear
 - Projective / Perspective
